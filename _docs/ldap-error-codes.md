# Common LDAP Error Codes

| Code | Description |
|------|-------------|
| 525 | User not found |
| 52e | Invalid credentials |
| 531 | Not permitted to logon at this workstation |
| 532 | Password expired |
| 534 | Account disabled |
| 701 | Account expired |
| 773 | User must reset password |
| 775 | User account locked |
